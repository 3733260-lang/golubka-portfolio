# Портфолио — Оксана Голубева

Статический сайт: `index.html` + `styles.css` + `script.js` + `assets/`. Без сборки, без зависимостей — открывается прямо в браузере.

## Как выложить на GitHub Pages (бесплатно)

1. Зарегистрируйся на [github.com](https://github.com), если ещё нет аккаунта.
2. Создай новый репозиторий, например `portfolio` (кнопка **New repository**). Публичный (Public).
3. Залей в него все файлы из этой папки — можно просто перетащить их в репозиторий через веб-интерфейс (**Add file → Upload files**), сохранив структуру папок (`assets/avatar.jpg` должен остаться внутри папки `assets`).
4. Зайди в репозитории в **Settings → Pages**.
5. В блоке **Build and deployment → Source** выбери **Deploy from a branch**, ветку `main`, папку `/ (root)`. Нажми **Save**.
6. Через 1–2 минуты сайт будет доступен по адресу вида:
   `https://твой-логин.github.io/portfolio/`

## Свой домен (если понадобится)

В **Settings → Pages → Custom domain** можно вписать свой домен (например, `oksana-design.com`) — GitHub сам выдаст инструкцию, какие DNS-записи добавить у регистратора домена.

## Как обновлять контент дальше

- Текст — прямо в `index.html`, ищи нужный блок по русскому тексту.
- Плейсхолдеры вида `[ дашборд CRM — заказы / аналитика ]` — это `<div class="shot">...</div>`. Меняешь на `<img src="assets/имя-файла.png" alt="...">`, картинку кладёшь в папку `assets`.
- Цвета и шрифты — в начале `styles.css`, в блоке `:root`.

## Локальная проверка перед загрузкой

Просто открой `index.html` двойным кликом — увидишь сайт как он есть, без сервера.
